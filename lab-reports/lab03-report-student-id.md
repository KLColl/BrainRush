### Опис реалізованих сторінок

#### Головна сторінка

Містить короткий опис проєкту BrainRush, його мету — розвиток когнітивних навичок користувача.

#### Сторінка "Про нас"

Сторінка містить інформацію про команду розробників і мету створення вебзастосунку BrainRush.
На ній представлено короткий опис концепції проєкту, список учасників команди з їхніми ролями.

#### Сторінка "Games"
Відображає список доступних ігор:
- Arithmetic
- Sequence Recall

- Tapping Memory

#### Сторінка "Profile"
Відображає статистику користувача:
- загальна кількість зіграних ігор;
- загальна кількість балів;
- фільтр за типом гри;
- таблиця зі статистикою рівнів та середнім часом проходження.

- Login (auth/login.html) — форма входу користувача.
- Register (auth/register.html) — форма реєстрації з перевіркою унікальності логіну та валідацією.

## Ключові фрагменти коду

### Маршрутизація в Flask

```python
from flask import Flask, render_template
from flask_login import LoginManager

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')
```
У реальному проєкті всі маршрути винесено у Blueprint-и, що дозволяє структурувати код за модулями:
```python
from app.routes.main import main_bp
app.register_blueprint(main_bp)
```
